Arrays

An array is a collection of elements of the same data type stored in consecutive memory locations. In Java, arrays are mutable but have a fixed size. Each element of an array can be accessed through its index.

Declaration and Initialization

Arrays can be declared using the following syntax:

dataType[] arrayName = new dataType[arraySize];

For example, to declare an integer array with five elements:

int[] arr = new int[5];

Unless specified, the default values for the elements in an array are 0 for integer, double, float, and long types, false for boolean types, and null for objects.

int[] arr = new int[5]; // creates an integer array with 5 elements
                System.out.println(arr[0]); // outputs 0

Accessing and Changing Elements

Elements in an array can be accessed and changed using their indices:

int[] arr = {1, 2, 3, 4, 5}; // creates an integer array with 5 elements
                System.out.println(arr[2]); // outputs 3
                arr[2] = 6;
                System.out.println(arr[2]); // outputs 6

Methods and Attributes

Arrays in Java have several methods and attributes, including:

  • length: returns the length of the array
int[] arr = {1, 2, 3, 4, 5}; // creates an integer array with 5 elements
                System.out.println(arr.length); // outputs 5

2D Arrays

A 2D array is an array of arrays. There are two types of 2D arrays: ragged arrays and rectangular arrays. A ragged array has rows of different lengths, while a rectangular array has rows of the same length.

// ragged array
                int[][] raggedArray = {{1}, {2, 3}, {4, 5, 6}};
                
                // rectangular array
                int[][] rectArray = {{1, 2}, {3, 4}, {5, 6}};

2D Arrays

A 2D array is an array of arrays. There are two types of 2D arrays: ragged arrays and rectangular arrays. A ragged array has rows of different lengths, while a rectangular array has rows of the same length. Another type of array that is related to 2D arrays is a parallel array. A parallel array is a set of two or more arrays that are related in some way; for example, two arrays could represent the x and y coordinates of a set of points.

// ragged array
                int[][] raggedArray = {{1}, {2, 3}, {4, 5, 6}};
                
                // rectangular array
                int[][] rectArray = {{1, 2}, {3, 4}, {5, 6}};
                
                // parallel arrays
                int[] xCoords = {1, 2, 3};
                int[] yCoords = {4, 5, 6};

Arraylists

An ArrayList is a part of the java.util package and is used to create dynamic arrays that can hold objects of one type. Unlike arrays, ArrayLists are mutable and can grow or shrink in size.

Declaration

ArrayList<dataType> arrayListName = new ArrayList<>();

For example, to declare an ArrayList of integers:

ArrayList<Integer> arrayList = new ArrayList<>();

Methods

ArrayLists have methods for accessing, changing, adding, and removing elements.

// adding elements
                ArrayList<Integer> arrayList = new ArrayList<>();
                arrayList.add(1);
                arrayList.add(2);
                arrayList.add(3);
                
                // accessing elements
                System.out.println(arrayList.get(0)); // outputs 1
                
                // changing elements
                arrayList.set(0, 4);
                System.out.println(arrayList.get(0)); // outputs 4
                
                // removing elements
                arrayList.remove(0);
                System.out.println(arrayList.get(0)); // outputs 2

Errors

One common error when using ArrayLists is an IndexOutOfBoundsException, which occurs when trying to access an element outside of the ArrayList's bounds.

Real-Life Examples

Arrays and ArrayLists are used in many real-life applications. For example, an array could be used to represent a list of temperatures throughout a day, with each element representing a temperature at a specific time. An ArrayList could be used to store a list of items in a shopping cart, with each element representing a different item.
