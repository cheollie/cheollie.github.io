overview:

What is OOP? Object Oriented Programming, or OOP, is a way of programming that focuses on representing real-world objects and concepts in code. Classes are descriptions of a kind of object or "blueprint", while objects are made from a class and have characteristics such as identity, state, and behavior.

  • encapsulation
    • using public and private to variables and methods for security
    • public: can be used inside and outside the class
    • private: can only be used inside the class
    • normally, variables are set as private and methods are set as public in order for variables to be incorrectly altered. however, helper methods, methods only used in other methods are usually private.

    This is why the methods and variables in our Hedgehog class are private or public!
