Encapsulation (+Access Modifiers)

  • using public and private to variables and methods for security
  • public: can be used inside and outside the class
  • private: can only be used inside the class
  • normally, variables are set as private and methods are set as public in order for variables to be incorrectly altered

parent class

  • base class, superclass, the ancestor

subclass

  • derived class, child class, descendant
  • inherits all public properties and methods of its parent class
  • can override methods of the parent
  • local variable
    • a variable only found only within a method
    • after method is executed it is destroyed
  • member data
    • data stored within an object or class
    • instance variables
      • member data within an instantiated object
  • when using superclass constructor, you must use super() in the first line in the subclass constructor

is-a/has-a

  • <subclass> is-a <parent class>
  • <class> has-a <variable/attribute>

abstraction

  • a general purpose class that contains common properties and methods

method types

  • mutator method/set method: can modify an object and its data
  • access methods/get methods: accesses content without changing and returns the value from the content
